## ⚙️ Instalação
```
# Abra um terminal e copie este repositório com o comando
$ git clone https://github.com/Elias-Ainsworth/Desafio-UploadImages.git
```

```
# Acesse a pasta da aplicação
$ cd Desafio-UploadImages
# Instale as dependências
$ yarn
# Inicie a aplicação
$ yarn dev
```
